Course Mappings

For courses where Moodle groups several activities into one Nova row (e.g. Assignments 1–6 average appears as "Assignment 1 total" in Moodle but as "Assignment 1" in Nova), specify the mapping here. Courses that use 1:1 name matching do not need an entry — leave them out entirely.

Key = Nova course name (partial match OK). Value = object mapping Nova row names to the exact Moodle column name that holds the grade for that row.

Example:
{ "Introduction to Computing": { "Assignment 1": "Assignment 1 total", "Assignment 2": "Assignment 2 total", "Quiz 1": "Quiz 1 total", "Quiz 2": "Quiz 2 total" } }
